What is the ROIC calculator?
The ROIC calculator is a tool that measures the return on invested capital.
How to calculate ROIC?
To calculate ROIC you should use the following formula:
NOPAT = EBIT * (1 - tax rate)
invested capital = Debt + Equity
ROIC = NOPAT / invested capital
Why use ROIC calculator?
The ROIC calculator is used in order to compute the return on invested capital.
